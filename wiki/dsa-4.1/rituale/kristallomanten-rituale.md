# Kristallomanten-Rituale (Achaz)

> **Quelle:** WdZ S. 139–141

## Überblick

Die Kristallomanten der Achaz binden ihre Magie an Edelsteine. Kernkonzept: Schliff des Steins bestimmt die begünstigte Zauberkomponente; Bindung lädt permanente AsP in den Kristall; je mehr pAsP gespeichert, desto stärker der Fokus.

---

## Schliff & Spontane Modifikationen

| Schliffart | Komponente | Erschwernis |
|-----------|-----------|-------------|
| Brillantschliff | Technik | +7 |
| Rosenschliff | Zauberdauer | +5 |
| Pendeloque (tropfenförmig) | Kosten | +6 |
| Cabochon (Rundschliff) | Zielobjekt | +2 |
| Tafelschliff | Reichweite | +3 |
| Smaragd-/Achteckschliff | Wirkungsdauer | +4 |

*Ein Madakristall des passenden Schliffs erlaubt einmalig diese Komponente ohne Erschwernis zu modifizieren.*

## Kernrituale

### Kristallbindung
| | |
|---|---|
| **AP** | 100 |
| **Probe** | KL / IN / FF (mod. nach Schliff + pAsP-Belastung) |
| **Kosten** | 13 AsP; 1 pAsP |
| **Wirkung** | Bindet Edelstein an Kristallomanten; macht ihn unzerbrechlich; stellt Selbstregenerationsstruktur her; ermöglicht RK-Fokus |
| **Mehrfach** | Kann mehrmals gesprochen werden; jede weitere Bindung = +1 pAsP, verstärkt Merkmals-Boni |
| **Fassungsvermögen** | Max. 1 pAsP pro 10 Karat; Mindere Kristalle: 1 pAsP pro 100 Karat |

### Kristallformung
| | |
|---|---|
| **AP** | 250 |
| **Probe** | KL / FF / KK (nach Schliff-Tabelle) |
| **Kosten** | 1 AsP / 100 Karat (mind. 5 AsP) |
| **Wirkung** | Ritualform von HARTES SCHMELZE; erhält Kristallstruktur; formt Edelsteine nach Wunsch (100 Karat ≈ 1 SR); kann Steine verschmelzen, Unreinheiten entfernen |

### Thesiskristall
| | |
|---|---|
| **AP** | 200 |
| **Probe** | KL / IN / FF (nach Kristallformungs-RkP*) |
| **Kosten** | (Qualität der Thesis) × W6 AsP |
| **Wirkung** | Fixiert Thesis als Lichtbild in Prismenkristall; bei Qualität ≥ 10: bewegtes 3D-Bild; kann als Lehrmeister für Zauber genutzt werden (gleiche Bedingungen wie geschriebene Thesis) |
| **Mindestgröße** | 2 × Qualitätspunkte je Karat pro Edelstein; Mindere Kristalle: 20 × Qualität |

### Madakristall
| | |
|---|---|
| **AP** | 300 (150 bei Merkmalskenntnis Kraft) |
| **Voraussetzungen** | RK ≥ 11, SF Kristallbindung, SF Matrixverständnis, Kugelzauber H'szints Auge |
| **Kosten** | Kein Ritual; Kristall muss im Zeitraum zwischen zwei Vollmonden jede Nacht dem Madalicht ausgesetzt sein |
| **Wirkung** | Potenter Kristall-Fokus; einmalig den entsprechenden Schliff als freie Spontane Modifikation nutzen (ohne Erschwernis) |
| **Haltbarkeit** | Bis zum nächsten Neumond (ca. 2 Wochen) |

### Matrixkristall
| | |
|---|---|
| **AP** | 200 |
| **Voraussetzungen** | RK ≥ 14, SF Thesiskristall |
| **Probe** | Zauberwurf + RK-Probe |
| **Wirkung** | Zauber wird +3 erschwert in Kristall gespeichert; bis zu RkP* Tage haltbar; Aktivierung = freie Aktion (RK-Probe MU/IN/CH); kein weiterer AsP-Aufwand; wie kurzlebiger APPLICATUS |

### Kristallkraft bündeln
| | |
|---|---|
| **AP** | 150 |
| **Probe** | MU / IN / CH |
| **Wirkung** | Gesamte Kraft eines Kristalls in einem einzigen Ritual verbrauchen; AsP = Gewicht in Karat (Mindere: ÷ 10); überschüssige AsP verstärken Wirkung oder erzwingen MR-Überwindung |
| **Risiko** | Über doppeltem RkW: unvorhersehbare Effekte (SL-Entscheid) |

## Schuppenbeutel (Achaz-Traditionsartefakt)

Der Schuppenbeutel wird aus der Haut der sechsten Häutung des Achaz gefertigt und ist sein Traditionsartefakt.

| Ritual | AP | Probe | Wirkung |
|--------|-----|-------|---------|
| Bindung des Schuppenbeutels | 100 | IN/FF/KO (+5) | Unzerstörbarkeit; Bindung; ermöglicht Apport |
| Ewige Wegzehrung | 200 | IN/FF/KO | Nahrung und Vorräte im Beutel bleiben haltbar |
| Suchende Finger | 100 | KL/FF/FF (+5+) | Bestimmte Gegenstände springen dem Griff entgegen; spart Aktionen bei Zaubervorb. |

---

## Key Takeaways

- Kristallbindung = Pflichtgrundlage; 1 pAsP pro Ritual; mehrfache Bindungen stapeln Merkmals-Boni
- Schliff-Tabelle bestimmt, welche Spontane Modifikations-Komponente der Stein unterstützt
- Madakristall erlaubt einmal pro Zauber eine freie Spontane Mod — muss vorher monatelang aufgeladen werden
- Matrixkristall = günstigerer Ersatz für APPLICATUS: kein pAsP nötig, aber begrenzte Haltbarkeit
- Kristallkraft bündeln ist hochriskant über doppeltem RkW
