# Ritualzauberei — Grundregeln

> **Quelle:** WdZ S. 105–108

## Was ist ein Ritual?

Rituale sind im Gegensatz zu Zaubern:
- **Länger** in der Ausführung (Minuten bis Tage)
- **Teurer** in AP (einmaliger Kauf; kein Steigern)
- **Traditionsgebunden** — nur bei Lehrmeistern der eigenen Tradition erlernbar (Ausnahme: einige allgemeine Objektrituale)
- **Probe auf Ritualkenntnis (RK)** statt ZfW; volle AsP auch bei Misslingen (außer Schamanenrituale: halbe AsP)

---

## Objektrituale

| Aspekt | Regel |
|--------|-------|
| Bindung | Einmalige Erschaffungsprobe bringt das Objekt in die magische Bindung |
| Aktivierung | Separat geprobt; nur der Besitzer kann aktivieren |
| Vollmondphasen | Manche Traditionen (Geoden, Kristallomanten) haben Mondphase-Abhängigkeiten |
| Permanente AsP | Erschaffung kostet oft 1+ pAsP, die permanent verloren sind |

---

## Allgemeine Objektrituale

Diese drei Rituale stehen mehreren Traditionen offen (Lernkosten und Beschreibungen wie angegeben):

### Apport
| | |
|---|---|
| **AP** | 200 |
| **Probe** | MU / IN / CH (+ 7) |
| **Wirkung** | Magisch gebundenes Objekt fliegt auf Befehl zurück; Reichweite 3 + RkP* Meilen; GS 15 |
| **Anwendung** | Wird auf ein Objekt (z.B. Hexenbesen, Schuppenbeutel) geworfen — aktivierbar durch Geste |

### Bannschwert
| | |
|---|---|
| **AP** | 200 |
| **Probe** | MU / MU / KL (+ 5) |
| **Wirkung** | Alle Beschwörungs- und Kontrollproben +1; +2 auf Bannungsproben; gilt als magische Waffe |
| **Tradition** | Gildenmagier, Kristallomanten u.a. |

### Druidenrache
| | |
|---|---|
| **AP** | 100 |
| **Probe** | MU / MU / KO |
| **Wirkung** | Konvertiert LeP + AuP + KO + CH + MU eines geschwächten Druidens in AsP; Wirkung = ZfW × 3; Dauer ca. 1 Stunde |
| **Tradition** | Druiden; auch in Druidentum-Büchern beschrieben |

---

## Ritualkenntnis-Probe

Für alle Rituale gilt die Ritualprobe auf Ritualkenntnis (RK) mit den bei dem Ritual angegebenen drei Eigenschaften:

```
RK-Probe: Eigenschaft 1 / Eigenschaft 2 / Eigenschaft 3 (± Modifikatoren)
```

Die **RkP*** bestimmen die Wirkungsstärke oder -dauer des Rituals.

---

## Schamanenrituale — Besonderheit

| Aspekt | Regel |
|--------|-------|
| Probensystem | Vier Ritualfertigkeiten statt einer einzigen RK: Geister rufen / bannen / binden / aufnehmen |
| AsP bei Misslingen | Nur halbe AsP (nicht volle) |
| Zeitpunkt | Mondphase und Tageszeit stark relevant (Tabelle WdZ S. 149) |
| Ritualplatz | Geweihter Ort erleichtert –4 bis –7; ohne Ritualplatz +3 |

---

## Key Takeaways

- Rituale = AP-Einmalkauf; keine Steigerung
- Ritualprobe auf RK; volle AsP auch bei Misslingen (außer Schamanen)
- Objektrituale haben Erschaffungs- und Aktivierungsprobe getrennt
- Apport, Bannschwert, Druidenrache sind traditionsübergreifende Rituale
- pAsP bei Erschaffung permanent weg — kann nicht zurückgekauft werden wie bei Zaubern
