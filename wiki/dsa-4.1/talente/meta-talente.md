# Meta-Talente

> **Quelle:** Wege des Schwertes (WdS) · S. 189–190 (Anhang 3)

## Was sind Meta-Talente?

Meta-Talente fassen mehrere Einzeltalente, die einem gemeinsamen Ziel dienen, zu einer einzigen Fertigkeit zusammen. Sie verkürzen häufig wiederkehrende Probenfolgen (z.B. tägliche Jagd im Wildnisabenteuer).

**Allgemeine Regeln:**
- Meta-Talente können **nicht direkt aktiviert oder gesteigert** werden.
- TaW = Formel aus den beteiligten Talenten; aber **nie mehr als das Doppelte des niedrigsten beteiligten TaW**.
- Ein Meta-Talent gilt als automatisch aktiviert, wenn alle Berechnungstalente aktiviert sind (soweit sie Spezialtalente sind).
- Spezialisierungen in Berechnungstalenten nur anrechenbar wenn sie für den Metatalent-Einsatz geeignet sind (Meisterentscheidung).
- Ob Meta-Talente verwendet werden, ist eine **Gruppenabsprache**.

## Pirschjagd (MU/IN/GE)

**TaW-Formel:** (TaW Wildnisleben + TaW Tierkunde + TaW Fährtensuchen + TaW Schleichen + TaW einer passenden Fernkampfwaffe) / 5

Bezieht sich auf Kleinwild (Vögel, Kaninchen etc.), nicht auf Großwild.

**Probe ablegen:** Jede Stunde auf der Pirsch; Meister kann nach Wilddichte erschweren.

| Bedingung | Modifikator |
|-----------|------------|
| Grunderschwert (mageres Revier) | nach Meisterwahl |
| Reichweite Waffe max. 20 Schritt | +7 |
| Reichweite Waffe bis 50 Schritt | +3 |
| Scharfschütze | –3 |
| Meisterschütze | –7 |
| Geländekunde passend | –3 |
| Ortskenntnis passend | –7 |

**Ergebnis:** Misslingt → Beute entkommen, Stunde verloren. Gelingt → TaP\* = Anzahl erbeuteter Tagesrationen.

## Ansitzjagd (MU/IN/GE)

Wie Pirschjagd, aber mit **Sich Verstecken** statt Schleichen in der Formel. Pro Probe 90 Minuten Zeitaufwand (statt 60).

## Nahrung Sammeln (MU/IN/FF)

**TaW-Formel:** (TaW Wildnisleben + TaW Sinnenschärfe + TaW Pflanzenkunde) / 3

In landwirtschaftlich genutzten Gegenden kann **Ackerbau** Pflanzenkunde ersetzen.

**Probe ablegen:** Einmal täglich nach mindestens einer Stunde Suche; pro Viertelstunde länger als eine Stunde: +1 (max. halber TaW).

In sehr lebensfeindlichen Gebieten (Eiswüste Gloranias) nicht möglich.

**Ergebnis:** Gelingt → 1 Tagesration + pro 3 TaP\* eine weitere.

## Kräuter Suchen (MU/IN/FF)

**TaW-Formel:** (TaW Wildnisleben + TaW Sinnenschärfe + TaW Pflanzenkunde) / 3

Verdoppeln der Suchzeit erhöht TaW auf das 1,5-Fache.

### Gezielte Suche

Suchschwierigkeit = Bestimmungswert der Pflanze + Verbreitungszuschlag:

| Verbreitung | Zuschlag |
|------------|---------|
| Sehr häufig | +1 |
| Häufig | +2 |
| Gelegentlich | +4 |
| Selten | +8 |
| Sehr selten | +16 |

Dauer: mindestens eine Stunde (doppelte Zeit: TaW ×1,5). Bei Erfolg: Grundmenge des Krauts + weitere Exemplare (je Suchschwierigkeit/2 aus den TaP\*).

### Zufällige Suche (was man findet)

TaP\* mit Verbreitungs- und Bestimmungszuschlägen aller Pflanzen der Region vergleichen (→ Zoo-Botanica Aventurica). Pro Kraut: 1 Grundmenge, dann Zuschlag der Pflanze von TaP\* abziehen, weitermachen bis TaP\* aufgebraucht.

## Key Takeaways

- Meta-Talente sind optionale Abkürzungen für häufige Wildnishandlungen.
- Der TaW kann nie über das Doppelte des schwächsten beteiligten Wertes steigen — breite Ausbildung ist entscheidend.
- Pirschjagd und Nahrung Sammeln liefern Tagesrationen (TaP\* = Anzahl).
- Kräuter Suchen referenziert die Zoo-Botanica Aventurica für Detailwerte.
- Geländekunde/Ortskenntnis reduziert die Schwierigkeit um 3 bzw. 7.
