# Metamagie (Zauberwerkstatt)

> **Quelle:** WdZ S. 38–45

## Überblick

Die **Zauberwerkstatt** erlaubt es, Spontane Modifikationen **permanent** in der Zaubermatrix zu verankern (Kleine Modifikationen) oder noch tiefgreifendere Eingriffe vorzunehmen (Große Modifikationen). Experten-Regelwerk: zeitaufwändig, nur in Ruhe/Abgeschiedenheit möglich.

## Voraussetzungen

| | Kleine Modifikation | Große Modifikation |
|--|--------------------|--------------------|
| ZfW in betroffenem Zauber | ≥ 10 | ≥ 15 (Übertragung Fremdrepräsentation: ≥ 12) |
| Talent Magiekunde | ≥ 10 | ≥ 15 (mit Spezialisierung: TaW 15(17)) |
| SF | Spezialisierung Zauberwerkstatt | Matrixverständnis |
| Merkmalskenntnis | ≥ 1 passende (oder Hauszauber) | ≥ 1 passende |
| Repräsentation | nur eigene (außer: Übertragung in eigene) | nur eigene |

## Modifikatoren auf die Proben

| Bedingung | Zuschlag |
|-----------|---------|
| Merkmalskenntnis Metamagie | –1 |
| SF Matrixverständnis | –2 |
| Spezialisierung Zauberwerkstatt | –2 |
| 2. passende Merkmalskenntnis | –2 |
| Je weitere Merkmalskenntnis | –2 |
| Hauszauber | –2 |
| Passende Begabung | –2 |
| Schlechte Bedingungen (z.B. Reisender) | bis +10 |
| Optimale Bedingungen (z.B. Puniner Akademie) | bis –2 |
| SF Matrixkontrolle | Gesamtzuschlag halbieren |

## Schrittweise Forschung

Modifikationen werden durch **Ansammeln von TaP* + ZfP*** über mehrere Schritte abgewickelt.

**Schritte berechnen:**
- Kleine Modifikation: Gesamtzuschlag ÷ 4 (mindestens 1)
- Große Modifikation: Gesamtzuschlag ÷ 2 (mindestens 2)

**Pro Schritt:**
- In jedem Schritt müssen TaP* und ZfP* in Höhe des Gesamtzuschlags angesammelt werden
- Pro Monat: **2 Proben** auf Zauber und 2 auf Magiekunde (Zauberwerkstatt), je mit halbem Zuschlag
- Übersteigt halber Zuschlag TaW/ZfW → gelungene Probe = 1 TaP*/ZfP*
- Pro Monat nur für **einen Schritt** akkumulierbar; überzählige Punkte verfallen

**Misserfolgsfolgen:**
- Eine Probe im Monat misslungen → alle Monats-Punkte weg
- Patzer oder alle 4 Monatsproben misslungen → von vorne beginnen

**Zeitbedarf:** ≥ 6 Stunden/Tag (3 ZE). Unterbrechung > 1 Woche/Monat → Monatspunkte weg. Unterbrechung > 1 Monat → je weiterer Monat ein Schritt zurück.

## Fixierung des neuen Zaubers

Nach Abschluss aller Schritte entstehen AP-Kosten:
- Kleine Modifikation: **AF × 20 AP**
- Große Modifikation: **AF × 40 AP**

*(AF = Aktivierungsfaktor = Grundkomplexität des neuen Zaubers; A=1, B=2, … H=8)*

Neuer Zauber erbt alle nicht veränderten Eigenschaften vom Vorgänger. ZfW des neuen Zaubers = niedrigster beteiligter ZfW – (Gesamtzuschlag ÷ 2), mindestens 0.

**Kostenveränderungen durch Forschung:**
- 1 misslungene Probe → Grundkosten +1W3–1 AsP (Kleine) / +1W6–1 AsP (Große)
- Neustart nötig → +1W3+1 AsP (Kleine) / +1W6+1 AsP (Große)
- Glücklich gelungene Probe → –1 AsP (Kleine) / –1W3 AsP (Große)

## Kleine Modifikationen

Permanent verankerte Spontane Modifikationen. Tabelle der Zuschläge:

| Modifikation | Zuschlag auf Proben |
|-------------|---------------------|
| Veränderte Technik (Aspekt verändern) | +4 pauschal + je +4 (verändern) / +8 (verzichten) |
| Veränderte Technik (zentral / traditionell) | +4 pauschal + je +8 (verändern) / +14 (verzichten) |
| Verfeinerte Technik (Komponente hinzufügen) | +4 pauschal + je +8 (ZD +5–10 %; AsP –10 %) |
| Verlängerte Zauberdauer (×2) | +4 pauschal + je +AF (AsP –5 %/Verdoppelung) |
| Verkürzte Zauberdauer (÷2) | +4 pauschal + je +(AF+4) (AsP +10 %) |
| Zauberwirkung erzwingen (1 Bonuspunkt) | +8 pauschal + je +6/Bonuspunkt (AsP +3/Punkt; ZD +1 Aktion) |
| Astralenergie einsparen (–10 % AsP) | +4 pauschal + je +(AF+6) (ZD ×2, außer als Große Mod.) |
| Zauber auf freiwillige Einzelperson | +12 (dauerhaft beschränkt: AsP –10 %) |
| Vergrößerung Reichweite (je Stufe) | +4 pauschal + je +8 (AsP +5 %/Stufe) |
| Verkleinerung Reichweite (je Stufe) | +4 pauschal + je +4 (AsP –5 %/Stufe) |
| Vergrößerung Wirkungsdauer (×2) | +4 pauschal + je +10 (AsP +20 %/Verdoppelung) |
| Feste Wirkungsdauer (A → fest) | +14 |
| Verringerung Wirkungsdauer (÷2) | +4 pauschal + je +6 (AsP –10 %) |

## Große Modifikationen

Schwerwiegendste metamagische Eingriffe; meist SF Matrixkontrolle erforderlich (Ausnahme: Übertragung Fremdrepräsentation).

| Modifikation | Zuschlag |
|-------------|---------|
| Veränderung des Wirkungsbereichs (Form) | +16 + AF (frei kontrollierbar: +24 + AF) |
| Minimierung Zauberdauer (–10 % ohne AsP-Erhöhung) | +8 pauschal + je +(AF+12) |
| Optimierung AsP-Kosten (–10 % ohne ZD-Erhöhung) | +8 pauschal + je +(2×AF+12) |
| Potenzierung der Wirkung (×2) | +8 pauschal + je +(AF+12) (AsP +80 %) |
| Erweiterung Ziele (freiwillig → beliebig) | +14 + 2×AF (AsP +10 %) |
| Erweiterung Ziele (Selbst → freiwillig Einzelperson) | +12 + AF |
| Elementare Transition (zu anderem Element) | +24 + 4×AF (Gegenelement: +6 extra) |
| Dämonische Transition | +28 + 4×AF |
| Metaphorische Transition (naheliegend) | +30 + 4×AF |
| Metaphorische Transition (entfernt) | +40 + 4×AF |
| Kombination zweier Zauber | +20 + doppelte Summe der AF (Simultanzaubern: –4) |
| Umkehrung via REVERSALIS | +18 + 4×AF (Probe auf beide Zauber) |
| Synthese zweier Zauber | +32 + doppelte Summe AF (+2 je Merkmal nicht in allen Ursprungszaubern; Matrixkontrolle Pflicht) |
| Übertragung in eigene Repräsentation | +40 + 4×AF (beide Rep. bekannt: +25 + 2×AF; schelmisch: +7) |
| Kombination von Repräsentationen | +30 + 3×AF |

**Transitionen — naheliegende Paare** (Metaphorische Transition): Herrschaft/Einfluss · Eigenschaften/Form · Objekt/Umwelt · Hellsicht/Verständigung.

## Repräsentations-Vor- und Nachteile (Zauberwerkstatt)

| Repräsentation | Vorteil | Nachteil |
|----------------|---------|----------|
| Borbaradianisch | kann 1 LeP → 2 AsP zaubern | nur Reichweite Berührung |
| Druidisch | Erzwingen-Variante möglich; nur halbe Punkte | kein Kontakt mit verarbeitetem Metall |
| Elfisch | Wirkungsdauer verdoppelt | Wiederholungen +5/Versuch |
| Geodisch | Kosten/Reichweite/Wirkungsdauer bei Elementarzaubern um 2 erleichtert | kein Kontakt mit verarbeitetem Metall |
| Satuarisch | nur 1/3 AsP-Kosten bei Misserfolg | Nachteil Erdgebundenheit |
| Kristallomantisch | Zauber kostet nur 3/4 AsP | Nachteil Kristallgebunden |
| Gildenmagisch | alle Modifikationen nur halb so erschwert | laute Formelaussprache |
| Scharlatanisch | Illusions-Modifikationen halb so erschwert; Illusionen doppelte WD | keiner |

## Freizauberei (Experten-NSC-Tool)

Spontaner magischer Effekt durch reinen Willen, ohne Formeln/Rituale. 1W6 bis 10W6 AsP je nach Mächtigkeit. Keine Regelvorschläge für Spielercharaktere — nur für: **Verhüllte Meister**, uralte Elfen/Geoden, fleischgewordene Magie-Wesenheiten (insgesamt maximal ein Dutzend Personen in Aventurien).

---

## Key Takeaways

- Kleine Modifikation = permanente Spontane Modifikation; Große = tiefgreifendere Eingriffe wie Transition, Synthese, Kombination
- Forschung = monate-/jahrelanger Prozess; reisende Zauberer haben +10 Erschwernis
- SF Matrixkontrolle halbiert Gesamtzuschlag — bei Großen Modifikationen oft entscheidend
- Kostenveränderungen bei Misserfolgen addieren sich dauerhaft auf den neuen Zauber
- Freizauberei ist reines Meister-Werkzeug für NSC-Übermächtige
