# Magie-Grundlagen

> **Quelle:** WdZ S. 5–10

## Kernkonzepte

| Begriff | Definition |
|---------|-----------|
| **Astralenergie (AE)** | Die aller Zauberei innewohnende Kraft; durchströmt die gesamte Welt Dere; auch Astralkraft, Sternenkraft, Sphärenkraft |
| **Astralpunkte (AsP)** | Spieltechnische Einheit der AE; werden bei Zauberhandlungen verbraucht, durch Schlaf regeneriert |
| **Permanente AsP (pAsP)** | Maximal speicherbare AE; sinkt, wenn AE dauerhaft in Artefakte oder Zauber eingebunden wird |
| **Astralleib** | Feinstoffliches Gewebe (Lebensseele) = Sitz der Persönlichkeit, Erinnerungen und astralen Muster |
| **Magisches Muster** | Einzigartiges astral-energetisches Gewebe jedes Wesens/Objekts; nur magische Wesen haben eines |
| **Thesiskern** | Die unveränderliche Kernwirkung eines Zaubers (unabhängig von Repräsentation) |
| **Repräsentation** | Traditionsgebundene Art, einen Thesiskern zu wirken — 9 Repräsentationen in Aventurien |
| **Merkmal** | Kategorisierung der Zauberwirkung (z.B. Beeinflussend, Schadenszauber, Form, Eigenschaft, Antimagie, Elementar, Dämonisch …) |
| **Komplexität** | Lernschwierigkeit eines Zaubers (unabhängig von Merkmalen/Repräsentation) |

## Wer kann zaubern?

**Zauberkundige** = alle Helden/MP die AE besitzen und/oder manipulieren können. Voraussetzung: Vorteil [[../vor-nachteile/vorteile|Vollzauberer]], [[../vor-nachteile/vorteile|Halbzauberer]] oder [[../vor-nachteile/vorteile|Viertelzauberer]].

## AE-Grundwert (Basis-AE)

Berechnung je nach Tradition (Beispiel Gildenmagier):  
`(MU + KL + IN + CH + KL + IN) / 2` + Vor-/Nachteile + Steigerung durch Große Meditation

**SF Gefäß der Sterne:** CH geht doppelt ein → `(MU+IN+CH+CH)/2`

## Regeneration von Astralpunkten

| Bedingung | AsP |
|-----------|-----|
| 6 Std. Schlaf (Grundregeneration) | 1W6 |
| Erfolgreiche IN-Probe danach | +1 |
| Vorteil Astrale Regeneration I / II / III | +1 / +2 / +3 |
| Nachteil Astraler Block | –1 |
| SF Regeneration I / II | +1 / +2 |
| SF Meisterliche Regeneration | Ersetzt W6 durch Leiteigenschaft/3; +3 statt Regen. II |
| Voller Ruhetag (2 Regenerationsphasen) | × 2 (wenn Meister erlaubt) |
| Komfortable Schlafstätte (Einzelzimmer) | +1 |
| Luxuriöse Schlafstätte (Suite) | +2 |
| Held erkrankt | insgesamt nur 1 AsP |
| Schlechtes bis extremes Wetter | –1 bis –5 |
| Schlechter Lagerplatz (Wildnisleben misslungen) | –1 |
| Ruhestörung / Wache gehalten | –1 |

Negative Ergebnisse = keine Regeneration. AE regeneriert maximal bis auf den Grundwert.

## Astrale Meditation (SF)

Umwandlung von LeP in AsP: `IN/CH/KO`-Probe, erleichtert um RkW/2 (oder TaW Musizieren/Selbstbeherrschung bei Elfen/Schelmen). Kostet 1 AsP zum Start; danach 1 LeP → 1 AsP pro SR, aber –1W3–1 LeP anschließend.

**Thonnys-Kraut:** ermöglicht auch ohne SF bis 2W6+2 LeP→AsP; erleichtert SF-Probe um 3, kein Start-AsP, kein LeP-Verlust danach.

## Permanente AsP (pAsP)

Manche Rituale und Artefaktherstellung verbrauchen **permanente** AsP → senken Basis-AE dauerhaft.
- **Rückkauf:** 50 AP pro pAsP (ohne weitere SF)
- **SF Matrixregeneration I:** 40 AP / 20 ZE
- **SF Matrixregeneration II:** 30 AP / 15 ZE (KL 18 erforderlich)

## Arten von Zauberei

| Art | Beschreibung |
|-----|-------------|
| **Spruchzauberei** | Kurzfristig; bewusster Willensakt + Zauberhandlung; Regeln → [[zauberregeln]] |
| **Ritualzauberei** | Langfristig; traditionsgebunden; Paraphernalia, Sternkonstellationen; Kosten immer voll |
| **Artefakte** | Verzauberte Gegenstände; Hohe Alchimie; auch von Nichtzauberern nutzbar |
| **Profane Alchimie** | Ohne rohe AE; Kräfte/Mischung der Zutaten |

## Key Takeaways

- Astralenergie ist Ressource (AsP) **und** strukturgebendes Medium (Muster) — beides gleichzeitig
- Misslingt ein Spruch: nur **½ AsP** Kosten (Ritual: volle Kosten)
- pAsP-Verlust durch Artefaktmagie ist häufig; Rückkauf kostet AP
- Hexen/Druiden/Geoden nennen AE „Erdkraft" / „Sumus Atem" — spieltechnisch identisch
