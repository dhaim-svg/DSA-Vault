# Hindernisse: Flussüberquerungen, Klettern und Schluchten

> **Quelle:** WdE S. 90–104

---

## Flussüberquerungen

In zivilisierten Gebieten: Fähren (gegen Gebühr, bis Kutschengröße) oder Brücken an Reichsstraßen. In der Wildnis: Furt suchen, Schwimmen oder Behelfsfloß.

### Methode 1: Furt (Durchwaten)

- Furt = Wassertiefe max. anderthalb Schritt
- Finden: Einheimischer-Tipp oder Phex-Gunst; breite Stellen tendieren zu flacher, aber tiefe Rinnen möglich
- Vorgehensweise: Tastend vorgehen, Gepäck auf Kopf oder Behelfsfloß; unbekannte Furt zuerst von gutem Schwimmer ohne Gepäck erkunden

**Probe (zu Fuß):** Pro angefangene 10 Schritt Flussbreite eine *Körperbeherrschungs-* oder *Akrobatik-Probe* (MU/GE/KK)

| Modifikator | Wert |
|-------------|------|
| SF Balance | –3 |
| SF Herausragende Balance | –7 |
| Wassertiefe knietief | –3 |
| Wassertiefe hüfttief | +0 |
| Wassertiefe brusttief | +3 |
| Wassertiefe bis zum Kinn | +6 |
| Strömung moderat | –3 |
| Strömung mittelstark | +0 |
| Strömung stark | +3 |
| Strömung reißend | +6 |
| Beladung unbeladen | –3 |
| Beladung maximal TK | +0 |
| Pro zusätzliche 0,5×TK Last | +1 |
| Untergrund trittfest | –3 |
| Kleinere Hindernisse (Steine) | +0 |
| Untergrund rutschig/sehr uneben | +3 bis +6 |

**Ergebnisse (zu Fuß):**
- **Patzer**: Knöchel verstauchend, Wasser schlucken (1W6 SP), Gepäck verloren, Riemen reißen; bei ≥ hüfttief + starke Strömung → mitgerissen werden
- **Mehr als 7 TaP fehlen**: Versinkt, Gepäck verloren; Schwimmen-Probe entscheidet wie viel geborgen wird
- **Probe misslingt**: Straucheln, Gepäck fällt ins Wasser und wird durchnässt
- **Probe gelingt**: Abschnitt problemlos durchquert

**Probe (reitend):** *Reiten-Probe*; geführtes Tier → *Abrichten-Probe* (Führender muss zusätzlich eigene Körperbeherrschungs-Probe schaffen)

Zusätzliche Reiten-/Abrichten-Modifikatoren:

| Modifikator | Wert |
|-------------|------|
| Tier wird geritten | +6 |
| Tier wird geführt | +0 |
| SF Geländehindernisse (Tier) | –3 |
| SF Trittsicherheit Gewässer (Tier) | –7 |
| Ungearbeitetes Tier | +3 |
| Unerfahrenes Tier | +0 |
| Erprobtes Tier | –3 |
| Geschultes Tier | –6 |
| Loyalität (LO) 0–5 | +3 |
| Loyalität (LO) 5–10 | +0 |
| Für je 2 LO über 10 | –1 |

**Ergebnisse (reitend):**
- **Patzer**: Tier stürzt schwer (2W6 SP), LO –3, Panik
- **>7 TaP fehlen**: Tier rutscht aus, Reiter fällt, Gepäck durchnässt; LO-Probe +3 oder Tier flieht
- **Probe misslingt**: Tier strauchelt; Reiter muss einfache Reiten-Probe schaffen, Tier verweigert zuerst

**Probe (fahrend):** *Fahrzeug Lenken-Probe*

| Modifikator | Wert |
|-------------|------|
| Wagen wird gefahren | +6 |
| Leittier wird geführt | +0 |
| Tiefe bis zu den Achsen | –3 |
| Tiefe bis Oberkante der Räder | +0 |
| Tiefer | +3 |
| Strömung wie oben | –3 bis +6 |
| Untergrund eben | +0 |
| Kleinere Hindernisse | +3 |
| Rutschig/sehr uneben | +6 bis +9 |
| Ausbildungsstand Zugtiere | +3 bis –6 |

**Ergebnisse (fahrend):**
- **Patzer**: Rad in Loch, Achse bricht, Wagen stürzt; Tiere verletzt (2W6 SP), Panik; Bergen zeitraubend
- **>12 TaP fehlen**: Rad versinkt, Wagen kippt, Ladung verteilt sich im Wasser
- **Probe misslingt**: Wagen hängt fest; Wiederholung nötig, +1 Erschwernis pro Fehlversuch

---

### Methode 2: Schwimmen

Hauptproblem: Empfindliches Gepäck (Schriftstücke, Zunder) und Gewicht. Kleidung/Textilien saugen Wasser; Taschen/Waffen ziehen in die Tiefe.

**Vorbereitung**: Gepäck Besserem Schwimmer anvertrauen oder Behelfsfloß (mind. armdicke Holzstücke fest vertäuen). Bei schmalem Fluss: Vorausschwimmer spannt Seil, restliche Ausrüstung wird hinübergezogen.

**Probe:** Pro angefangene 10 Schritt Schwimmstrecke eine *Schwimmen-Probe* (Strecke ≈ Flussbreite; bei Gegenströmung kann sich Strecke verdoppeln)

| Modifikator | Wert |
|-------------|------|
| Wellengang ruhig | –6 |
| Wellengang leicht | –3 |
| Wellengang stark | +0 |
| Wellengang reißend | +3 |
| Unbeladen | +0 |
| Maximal TK | +3 |
| Pro zusätzliche 0,5×TK Last | +1 |
| Behelfsfloß schieben/ziehen | +3 bis +6 |
| Dichter Unterwasserbewuchs | +3 |
| Gefährliche Felsen (starke Strömung) | +6 |

**Ergebnisse:**
- **Patzer**: 2W6 TP, in die Tiefe gezogen; *Schwimmen-Probe* pro KR nötig um aufzutauchen; 1 LeP pro KR unter Wasser
- **>7 TaP fehlen**: 1W6+2 TP an versteckten Hindernissen; *Selbstbeherrschungs-Probe* (erschwert um TP) ob Gepäck losgelassen wird
- **Probe misslingt**: Kurz untergetaucht, Wasser geschluckt, abgetrieben
- **Probe gelingt**: 10 Schritt problemlos durchschwommen

**Schwimmendes Reittier**: Reiten-Probe (Reiter) bzw. Abrichten-Probe (geführt) + separate Schwimmen-Probe des Tiers. Über-TK-Belastung → Tier geht unter, reagiert mit Panik (nur LO ≥ 12 kontrollierbar).

---

### Methode 3: Fähre

- In zivilisierten Gebieten, gegen Gebühr; transportiert bis Kutschen-Größe
- Keine Proben nötig (außer Verhandlung)

### Methode 4: Brücke (festgebaut)

- Nur an stark frequentierten Orten / Reichsstraßen
- In der Wildnis praktisch nicht vorhanden

### Methode 5: Behelfsmäßige Überquerung

- Behelfsfloß aus vertäuten Holzstücken
- Seiltechnik (Vorausschwimmer + Ziehseil)
- Nur bei schmalem Fluss praktikabel; zeitaufwendig aber sicherste Methode für empfindliche Güter

### Magische Hilfe bei Flussüberquerungen

- **SOLIDIRID**: Lichtbrücke (ZfW×15 Sekunden Wirkung in normaler Repräsentation)
- **WEICHES ERSTARRE**: Feste Schicht auf Wasseroberfläche (+10 auf Zauberprobe)
- **WELLENLAUF**: Über Wasser laufen; Stolperfalle bei Wellen
- **WASSERATEM**: Am Grund entlanglaufen
- **LEIB DER WOGEN**: Vor Ertrinken geschützt
- **ADLERSCHWINGE**: Tiergestalt zum Aufklären / Seil transportieren
- Efferd-Geweihte: *Begehen der Heiligen Wasser*
- Geodischer Wasserbann: Wasser verdrängen → trockener Fuß (Zonen-Variante nimmt Gefährten mit)

## Klettern und Schluchten

### Grundsituation

Felsspalten entstehen durch: Gebirgsbäche, Erdbeben, eingestürzte Höhlen (Zwergenbingen, Felsschratfraß), Magie (z. B. Felselementar). Sie schließen sich nicht von selbst.

**Drei Optionen**: springen, klettern oder Brücke bauen/finden

---

### Springen über Felsspalte

**Sprungweite eines Helden**: (GE+KK–BE) in Spann; *Athletik-Probe* gibt je TaP* +2 Finger zusätzliche Weite. Anlauf: mind. 5 Schritt Pflicht; ohne Anlauf: gesamte Weite halbiert; bei halbem Anlauf: ×0,75.

Modifikatoren:
- Überanstrengung: pro Punkt KK –1
- Kleinwüchsig: BE +1 / Zwergenwuchs: BE +2
- Ziel pro ½ Spann höher: +3; pro Spann niedriger: –1

**Ergebnistabelle Sprung (Held zu Fuß):**

| Ergebnis | Folge |
|----------|-------|
| Sprungweite reicht | Problemlos auf die andere Seite |
| Fehlt bis 1 Spann | Abrollen nötig; *Körperbeherrschungs-/Akrobatik-Probe* sonst 1W6 SP |
| Fehlt bis 2 Spann | Kante mit Beinen getroffen: 1W6 SP |
| Fehlt bis 3 Spann | Oberkörper liegt drüben: 1W6 SP; *Klettern-Probe –3* zum Hochziehen |
| Fehlt bis 4 Spann | Großteil Oberkörper drüben: 1W6 SP; *Klettern-Probe +3* nötig |
| Fehlt bis 1 Schritt | Nur Arme über Kante: 2W6 SP; *Klettern-Probe +6* zum Halten |
| Fehlt bis 6 Spann | Nur Fingerspitzen: 2W6 SP + *Klettern-Probe +6* zum Halten und +6 zum Hochziehen |
| Mehr als 6 Spann | Knall gegen die Wand: nur Klettern-Probe +6 rettet |

*Tipp: Steht jemand drüben, erleichtert seine Athletik-Probe (KK/KK/KK) die Klettern-Probe um die erzielten TaP*.*

### Sprung auf dem Rücken eines Reittiers

Zwei Proben: (1) *Reiten-Probe* (CH/MU/KK) zum Überreden des Tiers + (2) *Sprung-Probe* des Tiers.

Reiten-Probe-Modifikatoren:
- Loyalität: +10–LO
- Schlucht bis halbe Sprungweite: –3; bis Sprungweite: +0; pro Schritt darüber: +5
- Schlechte Eigenschaft Höhenangst: –Wert

Sprung-Probe-Modifikatoren:
- Pro 3 TaW Reiten des Reiters: –1
- Beladung bis halbe TK: +0; bis TK: +3; bis 1,5×TK: +6; bis 2×TK: +10

Ergebnis der Sprung-Probe:
- Fehlt bis 2 Schritt → Tier prallt gegen Wand, stürzt; Reiter rettet sich nur mit *Akrobatik-Probe*
- Fehlt bis 1 Schritt → Tier kommt ungeschickt auf; GE-Probe sonst 3W6 SP; Reiter 2W6+4 SP(A)
- Reicht aus → Problemlos

---

### Brücke bauen

#### Baum fällen

1. **Holzbearbeitung-Probe (IN/KL/KL)**: Geeigneten Baum auswählen; Misslingen → falscher Baum
2. **Holzbearbeitung-Probe (KL/IN/GE)**: Baum quer fällen; Misslingen → Baum stürzt ab und ist verloren
3. **Orientierungs-Probe** (für genaue Lage): Misslingen → Baum liegt schlecht/wackelt oder stürzt (≥7 TaP fehlen)

Alternativ: Baum aus der Umgebung schleppen (hohes Gewicht, Aufrichten nötig; Vorteil: Äste vorher absägen).

#### Überqueren per Balancieren

- **Probe**: Pro angefangene 10 Schritt *Akrobatik-Probe +7*
- Modifikatoren: Dicker Baum –7; dünner Baum –3; Äste +3; straff gespanntes Seil +0; leicht schwingend +3; stark durchhängend +7; auf allen Vieren (Baum) –7; mit straffem Halteseil –4; mit Balancierstange (2 m) –3, (4 m) –5; Balance SF –3; Herausragende Balance –7; Höhenangst +Wert
- **Patzer**: Sturz auf halber Strecke → Sicherungsseil rettet oder Fall
- **Misslingt**: Gleichgewicht verloren → kann ans Seil greifen und hangeln

#### Seil hangeln

- **Probe**: Pro angefangene 10 Schritt *Athletik-Probe*
- Modifikatoren: Straff gespannt –0; leicht schwingend +1; stark durchhängend +3; nur an Händen +5; Beine ums Seil +0; Höhenangst +Wert
- **>10 TaP fehlen / Patzer**: Sturz auf halber Strecke
- **Misslingt**: Kurz kraft los; Wiederholung +3 pro Fehlversuch

---

### Gruppenüberquerungen: Sicherungsmethode

- Immer wenigstens einen Helden sichern, bevor der nächste beginnt
- Wer drüben ist, kann Seil halten / straffen
- **Kettenprinzip**: Gepäck zuerst hinüberschieben (Floß am Seil); dann Personen einer nach dem anderen; letzte Person sichert sich selbst → Rückholseil empfohlen
- Tier über Behelfsbrücke: nur mit sehr trittsicheren Reit-/Lasttieren; Tier muss zu Sprung bewegt werden (Reiten-Probe)

### Sturzschaden (Referenz WdS S. 144)

- Stürze aus Schlucht / Felsspalte: voller Sturzschaden entsprechend der Fallhöhe
- Patzer beim Balancieren/Hangeln ohne Sicherungsseil → voller Absturz

---

### Magische Hilfe bei Schluchten

- **SOLIDIRID**: Überbrückt Fluss wie Schlucht
- **METAMORPHO GLETSCHERFORM**: Eisbrücke über Gletscherspalte
- **SPINNENLAUF**: Eine Seite hinunter, andere Seite hinauf klettern
- **KRÖTENSPRUNG**: Kurzabstand überbrücken
- **ADLERSCHWINGE**: Tiergestalt mit Flug- oder Kletterfähigkeit
- **ÄNGSTE LINDERN** / Segen der Heiligen Velvenya: gegen Höhenangst
- HASELBUSCH: Baum Ast wachsen lassen als improvisierten Steg
- **TRANSVERSALIS**: Wenn Zielort bekannt; zurückgekehrt dann Gefährten mitnehmen

## Key Takeaways

- Flussüberquerungen sind nach Methode unterschiedlich riskant: Furt (Probe pro 10 Schritt), Schwimmen (wie Furt aber Gepäck kritisch), Fahren (+6 durch Bewegung)
- Strömung ist der entscheidende Modifikator: von –3 (moderat) bis +6 (reißend)
- Sprung über Felsspalte: nur für Athleten; bereits 1 Spann Fehlweite erzwingt Klettern-Probe
- Brücke bauen kostet viel Zeit (2–3 Proben); auf der Flucht nicht praktikabel
- Gruppenüberquerungen: immer Seil sichern, Kettenprinzip, Rückholseil
- Temperaturskala für CALDOFRIGO: 17 Stufen von Wertigkeit 0 (< –150°C) bis 150 (> 1.500°C)
