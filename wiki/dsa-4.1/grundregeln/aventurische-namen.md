# Aventurische Namen

> **Quelle:** Wege der Helden, Anhang 3, S. 307–313

Übersicht der Namensgebungstraditionen aventurischer Völker und Kulturen. Die Namenslisten dienen gleichermaßen Spielern (Heldenname) und Spielleitern (Meisterpersonen).

## Kulturen und ihre Benennungstraditionen

| Kultur / Region | Sprache/Herkunft | Besonderheiten |
|----------------|-----------------|----------------|
| **Garetisches Mittelreich** | Garethi | Vor- + Nachname; Nachname aus Beruf, Herkunft oder Eigenschaft (Steinhauer, Angbarer); Adel mit mehreren Vornamen und Lehen (von, zur, zu) |
| **Albernia** | Isdira-Garethi-Mix | Ähnlich Garethi; keltisch-klingende Varianten; Adel mit ui (Sohn) / ni (Tochter) + Vorfahren-Name (Aedin ui Burrigh) |
| **Almada** | Garethi / Bosparanisch | Dreiteiliger Name: Eigen-, Bospara-Schmuckname, Familienname; Koseform mit -ela/-elo |
| **Andergast** | Garethi | Nur Freibauern/Städter tragen Familiennamen; Frauen hängen -in an Mannsnamen |
| **Bornland** | Garethi + Norbardisch | Stark norbardisch geprägt; slawisch klingende Varianten (-jan, -jew, -ow) |
| **Nostrien** | Altertümliches Garethi | Archaischer Klang; Wohlhabende ändern gerne Namen |
| **Weiden** | Garethi | Vorliebe für Bär-, Fee-, Ron-/Rond-Vorsilben; antiquierte Endungen |
| **Horasreich** | Horathi (Garethi-Dialekt) | Weicher, melodischer Klang; Adelstitel ya (z.B. Nepolemo ya Torese); de/di/du statt von |
| **Sydlande / Brabaci** | Brabaci | Lange, blumige Mehrfachnamen; Rufname + Koseform (-ita/-eto); männlich -o, weiblich -e/-is |
| **Thorwal** | Hjaldingsch | Zweisilbige Vor-Namen mit Tugend-Bedeutung (Tor- = Mut/Held; -kir = Kämpfender); Beiname für Heldentaten |
| **Gjalskerland** | Hjaldingsch | Vater-/Mutternamens-Tradition mit bren (Sohn des) / brai (Tochter der); Schamanen übernehmen Mentornamen (dur) |
| **Fjarninger** | Thorwalsch-nah | Ähnlich Thorwalsch; Kampf-/Ehrennamen beliebt; keine Familiennamen |
| **Nivesen** | Puka-Puka | Rufname + Stammhäuptling ("Ich bin Nirka von Einuks Stamm"); geheimer Tapamname; Wolfsnamen |
| **Norbarden** | Norbardisch | Sippen- statt Familienname; weibl./männl. Namen sehr ähnlich; Endung -a/-ja (w) vs. -o/-jo (m) |
| **Tulamiden** | Tulamidya | Eigenname + ibn/saba (Sohn/Tochter des) + evtl. sâl (Schüler des) + ay (aus) + al' (Ehrenname) |
| **Novadis** | Tulamidya-Variante | Wie Tulamiden; voller Name selten genannt (Blutrache-Schutz); Stammesname mit ben/saba |
| **Aranien** | Garethisch-Tulamidisch | Dreiteiliger Mischname (Eigenname, Vatername, Familienname); tulamidische Basis + garetische Variationen |
| **Ferkinas** | — | Eigenname + iban (Sohn) / sabu (Tochter) + Vatername; Stammesname mit Bân |
| **Trollzacker** | Zulchammaqra | Schriftlos; dunkler, rauer Klang; kein Vater-/Muttername; Ehr- oder Schandname möglich |
| **Zahori** | Tulamidisch/Almadisch | Tulamidische Vornamen + almadisierte Varianten; Sippennamen (Alfahan, Cruento etc.) |
| **Waldmenschen** | Puka-Puka | Selbst gewählter Erwachsenenname nach Tat/Vision; geheimer Tapamname; Utulu-Namen länger und rhythmisch; nicht geschlechtsspezifisch |
| **Elfen** | Isdira | Erstname (einmalig, lyrisch-bedeutsam) + Zweitname (Familienattribut oder persönlicher Beiname, kann sich ändern); typische Elemente: Naturwörter (Mondglanz, Winterrufer) |
| **Zwerge** | Rogolan | Söhne nach Vater, Töchter nach Mutter (groscho/groschna); Alliteration oder Reimform beliebt; Hügelzwerge mit menschlichen Nachnamen |
| **Orks** | Orkisch | Kurze, harte Vornamen; Kriegs-/Ehrennamen nur für außergewöhnlich tapfere Orks; Svelttal-Orks kombinieren Ork-Vornamen mit garethischen Nachnamen |
| **Goblins** | Goblinisch | Kurze, guttural-klingende Namen; Zweitname für heldenhafte Taten (selten); Festumer Goblins mit Mischformen |
| **Achaz** | Rssahh | Für Menschen nicht nachzuahmen; Name wird erst mit 6. Häutung vergeben; zuvor Rufname aus Eigenschaften; nicht geschlechtsspezifisch |
| **Zyklopäische Inseln** | Zyklopäisch | Lange Namen mit y, ph, ë, ï, ÿ; Adel mit a' + Wohnort + dylli + Familiensitz |
| **Maraskan** | Garethi/Tulamidya | Garethi oder Tulamidisch + Suffixe -eran/-jian/-mold (m), -jid/-yscha (w); Herkunftsort oft als Zusatz |

## Hinweise für Spieler

- Aventurische Namen klingen **anders** als irdische oder andere Fantasy-Welten — kein Drizzt, Gandalf oder Conan
- Viele Namen tragen eine Bedeutung (Tugend, Beruf, Ahnen), die Spieler ins Konzept einweben können
- In hinterwäldlerischen Regionen: Familiennamen sind unüblich; Beruf oder Herkunftsort als Zusatz
- Kulturmischungen (z.B. Bornländer mit norbardischen Vorfahren) ergeben oft interessante Namensformen

## Key Takeaways

- Namensgebung ist ein Spiegel der Kultur: tulamidische Ehrennamen, elfische Naturpoesie, orkische Kürze
- Bei der SL-Verwendung für Meisterpersonen: Ein einprägsamer, auch etwas kauziger Name ist wertvoller als ein perfekter
- Die vollständigen Namenslisten (S. 307–313 im Original) enthalten hunderte Beispiele pro Kulturraum
