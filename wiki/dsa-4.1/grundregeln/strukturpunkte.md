# Strukturpunkte

> **Quelle:** Wege des Schwertes (WdS) · S. 191 (Anhang 5)

## Konzept

Unbelebte Objekte haben drei Werte:

| Wert | Bedeutung |
|------|----------|
| **Strukturpunkte (StP)** | Gesamtschadenkapazität — wie viel ein Objekt aushält |
| **Härte (H)** | Widerstand gegen Schaden (wirkt wie RS) |
| **Struktur (S)** | „Bruchfaktor" — Maß für inneren Zusammenhalt (Skala 0–21) |

## Strukturpunkte — Beispielwerte

| Objekt | StP (ca.) |
|--------|-----------|
| Glasphiole | 1 |
| Irdener Teller | 3 |
| Seil | 10 |
| Einfache Brettertür | 30 |
| Stabile Außentür, Truhe, Marmorstatue | 50 |
| Mauerstücke (je nach Dicke) | 100+ |

## Härte — Richtwerte

| Material / Konstruktion | Härte |
|------------------------|-------|
| Stoffe, Feinglas, edle Keramik | 0 |
| Einfache Innentür, Möbel, Fensterladen | bis 6 |
| Massive Steineiche (Truhen, Burgluken) | 12 |
| Einfachste Ziegelkonstruktion | 12 |
| Stabile Feldsteinmauer | 25 |
| Eisenplatten auf Holz | 20+ |
| Eisen-/Bronzeobjekte (Ketten) | 15–20 |
| Granit, Basalt, Zwergenguss, Stahlplatten | 30+ |

## Strukturwert (S) — Beispiele

| Objekt | S |
|--------|---|
| Kartenhaus (sprichwörtlich) | 0 |
| Dünne Brettertür | 8 |
| Fenster (Ölpapier) | 6 |
| Fenster (Glas) | 8 |
| Fesseln — Seile | 6–15 |
| Fesseln — Ketten | 12–20 |
| Massive, verstärkte Ausfallspforte | 18 |
| Glasbehälter | 2–8 |
| Metallbehälter | 5–18 |
| Waffe | 18 – aktueller BF |
| Rüstung (Stoff) | 6 |
| Rüstung (Leder) | 8 |
| Rüstung (Kette/Schuppe) | 12 |
| Rüstung (Massivplatten wie Kürass) | 15–18 |
| Stahlkugel (massiv) | 20 |
| Regeltechnisch unzerbrechlich | 21 |

## Schaden an Objekten

**Gewöhnlicher Waffeneinsatz:**
> TP der Waffe − Härte des Objekts = StP-Verlust

**Zauber (Merkmal Schaden):** Ignoriert die Härte.

**Telekinetische Manipulation** (MOTORICUS, KLICKERADOMMS): Nur halbe Härte überwinden.

**Elementarer Sekundärschaden:** Elementspezifische Regeln (→ WdS S. 123).

**Faustformel Rüstungen:**
- StP ≈ RS × 10
- Härte ≈ RS × 2

## Strukturproben

Wenn StP unter bestimmte Schwellen sinken, wird eine Strukturprobe (1W20 gegen Strukturwert S) fällig:

| Schwelle | Probe |
|---------|-------|
| StP < ½ ursprünglicher StP | 1 Strukturprobe; Misslingen = Objekt zerstört |
| StP < ⅓ ursprünglicher StP | weitere Probe |
| StP < ¼ ursprünglicher StP | Probe bei jedem weiteren Schaden |

## Tür einrennen (vereinfacht)

KK-Probe, erschwernis = Härte der Tür. Übrige KP* senken StP der Tür. Jeder misslungene Versuch kostet den Einrennenden 1W6 SP(A).

## Belagerungsgerät

Eigene Schadensscala: **TP\*** statt TP.

> TP\* − ⅓ Härte = SP\*-Verlust an „Baupunkten" (= ⅒ der StP des Bauelements)

Strukturwert des Bauelements wird wie bei Handwaffen gehandhabt. Als Bauelemente gelten vollständige Konstruktionen (Torturm, Mauerabschnitt, Mast, Geschütz).

## Key Takeaways

- Härte wirkt wie RS: nur TP oberhalb der Härte schaden dem Objekt.
- Strukturproben werden fällig, sobald StP unter ½, ⅓, ¼ des Maximalwerts fallen.
- Rüstungs-Faustformel: StP ≈ RS × 10, H ≈ RS × 2.
- Regeltechnisch unzerstörbar = S 21 (z.B. magische Artefakte).
