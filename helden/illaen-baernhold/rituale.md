# Rituale — Illaen Baernhold

**Ritualkenntnis:** Gildenmagie TaW **11** (Voraussetzung für alle Stabzauber)

## Stabzauber (9 Rituale)

Alle Stabzauber sind an Illaens gebundenen Magierstab geknüpft. Aktivierung = freie Aktion. Details → [[wiki/dsa-4.1/rituale/stabzauber|Stabzauber & Kugelzauber]].

| Stabzauber | Effekt (Kurzform) |
|------------|------------------|
| Stabzauber: Bindung | Grundbindung; Voraussetzung für alle weiteren Stabzauber |
| Stabzauber: Fackel | Stab leuchtet auf Kommando |
| Stabzauber: Hammer des Magus | Stab als magische Waffe (erhöhte TP) |
| Stabzauber: Kraftfokus | −1 AsP auf jeden Zauber |
| Stabzauber: Merkmalsfokus | −1 auf Proben eines Merkmals (gewählt: ?) |
| Stabzauber: Modifikationsfokus | +1 spontane Modifikation pro Zauber |
| Stabzauber: Seil (des Adepten) | Stab als magisches Kletterseil |
| Stabzauber: Stabverlängerung | Stab verlängert sich auf Befehl |
| Stabzauber: Zauberspeicher | Speichert Zauber (1 Vol / 20 AsP) |

*Hinweis: Im Heldendokument „Stabzauber: Fackel" — in stabzauber.md heißt das Ritual „Ewige Flamme". Hier die Bezeichnung aus dem Heldendokument übernommen.*
